### What Is 

**React в тезисах** <br>
-> React - JS-библиотека для отрисовки пользовательских интерфейсов (user interfaces, UI)<br>

-> шаблонизаторы отделяют View от всего остального<br>
-> слой View в [[MVC]]<br>
